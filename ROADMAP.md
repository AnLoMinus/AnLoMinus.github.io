# מפת דרכים · AnLoMinus Studio

מסמך זה מרכז את תכנון הגרסאות הקרובות בדרך להפיכת האתר לפלטפורמה בלוגיסטית־קהילתית מלאה בסגנון ניאון מיסטי.

## v1.1.0 · Neon Flux Update (דצמבר 2025)
- רענון חזותי חשמלי: שכבות ערפל, זוהר אנרגטי וטיפוגרפיה מרובת-משקלים.
- Section בלוג ראשוני: Hero לפוסטים עדכניים, רשימת קטגוריות וקריאה לניוזלטר.
- חיבור Markdown מבוסס Git (תיקייה `posts/`) עם תבניות לפרסום מהיר.
- בסיס נתוני מדדים (YAML/JSON) להצגת KPI בפרויקטים סימולטיביים.

## v1.2.0 · Quantum Blog Release (פברואר 2026)
- אינדקס פוסטים עם חיפוש, תגיות, RSS וניווט לפי סוגי תוכן (Field Notes / Capsules / Signals).
- מצבי Playbook: רכיבי Accordion ומדדי Response המיועדים לתרחישי סייבר.
- שילוב Storyblok/MDX לעריכה עשירה + פריוויו חי בתוך הממשק.
- הרחבת האנליטיקה: ספירת צפיות, זמן קריאה ומדד אנרגיית קהילה (Engagement Score).

## v1.3.0 · Community Mesh (רבעון שני 2026)
- דשבורד אינטראקטיבי המאגד נתוני GitHub, TryHackMe, Discord ו-Telegram.
- לוח אירועים עם RSVP, חיבור ל-Zoom/Calendly ו-API קטן לשיתוף נתונים.
- מקטע Workshops חי הכולל טפסי הרשמה, תיעוד הקלטות וספריית משאבים.
- פיילוט ל-AI Assistants: המלצות קריאה, המשך כתיבה לפוסטים והצעות שיתופי פעולה.

## מעבר ל־v1.3.0 (חזון 2026+)
- אפליקציית מובייל היברידית (React Native / Expo) לצריכת הבלוג וקהילה בתנועה.
- אינטגרציית Real-Time Chat עם אימות מאובטח (Supabase / Firebase + OTP).
- שילוב API חיצוניים לניתוחי סיכונים אוטומטיים ו-Bug Bounty פנימי.
- מערכת Badge-ים להוקרת תרומות חברי קהילה ומשובים בזמן-אמת.

> המסמך יתעדכן עם כל החלטה חדשה ויקושר לעדכוני ה־`CHANGELOG`. שמירה על סינכרון בין התיעוד לקוד תבטיח שקיפות מלאה בתהליך הפיתוח.
